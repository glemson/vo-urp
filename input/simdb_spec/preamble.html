Abstract

This document gives a detailed description of the Simulation Data Model. It lays out its packages with their dependencies and defines all the object types, data types and enumerations with all their interrelations. It gives a list of all UTYPEs, cross linked to their definition in the document.

This document accompanies the "Simulation Data Model" working draft which gives a general overview of the model, its background, goals and issues. That document is available under the following link: http://volute.googlecode.com/svn/trunk/projects/theory/snapdm/specification/SimulationDataModel.0.9.doc. This document is derived from a UML diagram, designed using MagicDraw Community Edition 12.1, serialised to XMI in http://volute.googlecode.com/svn/trunk/projects/theory/snapdm/specification/uml/SimDM_DM.xml. A PNG version of that model can be found here.
