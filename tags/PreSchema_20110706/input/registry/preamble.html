Abstract

This document gives a detailed description of a data model for an IVOA Registry. It lays out its packages with their dependencies and defines all the object types, data types and enumerations with all their interrelations. It gives a list of all UTYPEs, cross linked to their definition in the document.

This document provides a UML version of the Resource Data Model that so far existed mainly as an XML schema. The goal of this document is to provide a generic version of that model in UML and of derived representations, most importantly a relational version.
