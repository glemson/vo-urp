---------------------------------------
   JDBC drivers
---------------------------------------


This folder contains the JDBC drivers for Java 1.4+ and are Type 3 or 4 (no native library needed == full java driver) :

- postgresql-8.3-603.jdbc3.jar      PostgreSQL JDBC Type 3 driver for version 8.x
- sqljdbc-2005-1.2.jar              Microsoft SQL Server 2005 driver
